# 🔑 Google Maps APIキー取得マニュアル

会長、アプリの設定画面にある「Google Maps APIキー」の取得手順をまとめました。
このキーがあることで、店名から住所や価格帯を自動取得できるようになります。

---

## 📅 おおまかな流れ
1. **Google Cloud Console** にログイン
2. プロジェクトを作成（または選択）
3. **Places API** を有効化
4. **認証情報（APIキ⁠ー）** を作成

---

## 🛠️ 詳細手順

### 1. Google Cloud Console へアクセス
[Google Cloud Console](https://console.cloud.google.com/) にアクセスし、Googleアカウントでログインします。

### 2. プロジェクトの作成
- 画面上部のプロジェクト選択（または「プロジェクトを作成」）から、新規プロジェクトを作成します。
- 名前は `Sakekasu-App` など分かりやすいものでOKです。

### 3. APIの有効化
- 左上のメニュー（≡）から **「APIとサービス」 > 「ライブラリ」** を開きます。
- 検索窓に **「Places API」** と入力し、出てきたものをクリックして **「有効にする」** を押します。

### 4. APIキーの発行
- メニューから **「APIとサービス」 > 「認証情報」** を開きます。
- 画面上部の **「+ 認証情報を作成」 > 「APIキー」** をクリックします。
- **「APIキーを作成しました」** と表示され、`AIzaSy...` で始まる文字列が表示されます。これがAPIキーです！

---

## ⚠️ 重要な注意点（セキュリティ）

> [!IMPORTANT]
> **「キーを制限」の設定を推奨します**
> 生成されたキーの横にある「Edit（編集）」から、以下の制限をかけると安全です。
> - **API の制限**: 「Places API」のみで使用できるように設定。
> これにより、万が一キーが漏洩しても他のサービスを悪用されるのを防げます。

---

## 📱 アプリへの設定
取得した `AIzaSy...` の文字列をコピーし、アプリの設定画面の入力欄に貼り付けて **「保存」** を押せば完了です！

何か操作で迷うことがあれば、すぐにマナ（またはエンジニアのミレイ）を呼んでくださいね。✨
