# 新商品企画案：沖縄の熱狂を食卓に。エリソース＆沖縄スパイスお試しセット

## 1. コンセプト

「見つからない」問題を解決するため、まずはAmazonで検索されやすい「沖縄」「調味料」という切り口で、中毒性の高いエリソースを体験してもらうためのエントリーモデル（お試しセット）を構築します。

## 2. 商品内容

- **エリソース・シークヮーサー（1本）**: メイン商品。
- **沖縄流ちょい足しスパイス（小袋）**: 「味変」を楽しんでもらうためのリサ厳選スパイス。
- **エミ特製「エリソース・レシピカード」**:
  - サーモン寿司への絶妙なかけ方
  - 豚しゃぶ・冷やしうどんへの応用
  - キャンプ飯での活用術

## 3. 販促・グロース戦略（エミ）

- **Amazon SEO**: 「沖縄 調味料」「シークヮーサー ソース」「サーモン 刺身」のキーワードで待ち伏せ。
- **SNS連動**: 会長のInstagramで、エリソースをかけた瞬間の「シズル感」重視のショート動画を公開。
- **同梱物**: 次回購入時に使えるクーポンコードと、NOTEでの開発裏話へのQRコード。

## 4. 収益性・物流（リサ）

- **目標価格**: 1,980円（税込・送料込）※お試ししやすい価格帯
- **想定利益率**: 32% (広告費・物流費を差し引いた実質利益)
- **発送元**: FBA（Amazonフルフィルメント）を推奨。
